Contest Rules and Info

  1. Peer judged foto contest.
  2. Users must join contest to be able to participate beyond viewing fotos.
  3. The top three vote count fotos for the currently selected theme are highlighted on the home page.
  4. One vote per foto per user.
  5. Users can not vote for their own fotos.
  6. Users may add or delete their own fotos at any time.
  7. If there is a tie vote count in the currently selected theme for a foto to be highlighted on the home page the foto chosen is arbitrary.
